# دليل المستخدم - G777 Ultimate
## كيفية استخدام نظام إدارة العملاء (CRM)

مرحباً بك في **G777 Ultimate**! هذا الدليل سيشرح لك كيفية استخدام الميزات الرئيسية بطريقة بسيطة.

---

## 1. لوحة التحكم الرئيسية (Dashboard)

عند فتح البرنامج، ستظهر لك لوحة التحكم التي تعرض:

| العنصر | الوصف |
|--------|-------|
| **إجمالي العملاء** | العدد الكلي للعملاء المسجلين |
| **عملاء محتملون (Leads)** | العملاء الجدد الذين لم يشتروا بعد |

---

## 2. فهم ألوان البطاقات

البرنامج يستخدم **تصميم النيون (Cyberpunk)** مع ألوان مميزة:

### ألوان أنواع العملاء:
| اللون | المعنى |
|-------|--------|
| 🟢 **أخضر نيون** | عميل محتمل (Lead) - لم يشترِ بعد |
| 🔵 **أزرق نيون** | عميل فعلي (Customer) - اشترى منك |
| 🟣 **بنفسجي نيون** | عميل VIP - عميل مميز |

### ألوان الحالة:
| اللون | المعنى |
|-------|--------|
| ✅ **أخضر** | متصل / نشط |
| 🟡 **أصفر** | في الانتظار |
| 🔴 **أحمر** | غير متصل / خطأ |

---

## 3. كيفية فلترة الأرقام

### خطوات الفلترة:

1. **افتح صفحة العملاء** من القائمة الجانبية
2. **اختر نوع الفلتر** من القائمة المنسدلة:
   - `كل العملاء` - عرض الجميع
   - `عملاء محتملون (Leads)` - فقط الجدد
   - `عملاء فعليون` - من اشتروا
   - `VIP` - العملاء المميزون

3. **فلترة حسب المدينة** (اختياري):
   - اكتب اسم المدينة في خانة البحث
   - مثال: "القاهرة" أو "الإسكندرية"

4. **فلترة حسب الاهتمامات** (اختياري):
   - اختر واحد أو أكثر من:
     - تسويق
     - عقارات
     - سيارات
     - إلكترونيات

### نصيحة:
> يمكنك دمج أكثر من فلتر معاً. مثلاً: عملاء محتملون + القاهرة + مهتمون بالعقارات

---

## 4. كيفية تصدير ملفات CSV

### خطوات التصدير:

1. **اذهب لصفحة العملاء**
2. **طبق الفلاتر المطلوبة** (اختياري)
3. **اضغط على زر التصدير** (أيقونة 📥)
4. **اختر مكان الحفظ** على جهازك
5. **افتح الملف** في Excel أو Google Sheets

### محتويات ملف CSV:

| العمود | الوصف |
|--------|-------|
| `name` | اسم العميل |
| `phone` | رقم الهاتف |
| `city` | المدينة |
| `customer_type` | نوع العميل (lead/customer/vip) |
| `interests` | الاهتمامات (مفصولة بـ ;) |

### مثال:
```
name,phone,city,customer_type,interests
أحمد محمد,+201234567890,القاهرة,lead,تسويق;عقارات
سارة أحمد,+201987654321,الإسكندرية,customer,سيارات
```

## 5. إضافة علامات (Tags) للعملاء

### لإضافة اهتمام جديد لعميل:

1. **اضغط على بطاقة العميل** لفتح التفاصيل
2. **اضغط على أيقونة التعديل** (✏️)
3. **اكتب العلامة الجديدة** في خانة "الاهتمامات"
4. **اضغط حفظ**

### أمثلة على العلامات:
- `عقارات`
- `سيارات فاخرة`
- `استثمار`
- `VIP`
