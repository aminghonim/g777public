## 🚀 Key Features

- 🎨 **Premium Flutter UI** - Multi-platform support (Web/Desktop) with Cyberpunk aesthetic.
- ⚡ **FastAPI Backend** - High-performance asynchronous REST API.
- 🧪 **Modular Architecture** - Satisfies strict G777 constraints for Zero-Regression and Surgical Fixes.
- 🧠 **AI Integration** - Powered by Google Gemini 2.0 for smart responses and automation.
- 🔒 **Enterprise Monitoring** - Production-ready Sentry integration and centralized logging.
- 🐳 **Docker Orchestration** - Fully containerized multi-service environment (PostgreSQL, Redis, Evolution API).

## 🛠️ Stack Reference

- **Backend**: Python 3.11+ (FastAPI)
- **Frontend**: Flutter (Dart)
- **Engine**: Evolution API (Baileys)
- **Database**: PostgreSQL / Supabase
- **Monitoring**: Sentry SDK
- **Environment**: Docker & Docker Compose

## 📦 Installation

### Prerequisites

- Python 3.11+
- Flutter SDK (latest stable)
- Docker & Docker Compose

### Quick Start (Backend)

```bash
# Install dependencies
pip install -r requirements.txt

# Start backend server
python main.py --dev
```

### Quick Start (Frontend)

```bash
cd frontend_flutter
flutter pub get
flutter run -d chrome
```
